Módulo para validação de documentos brasileiros.

Description

Módulo para validação de documentos brasileiros. Atualmente suporta validação de CPF com ou sem pontuação, e modo estrito opcional.

Exemplos: > despachante:valid(cpf, "825.205.780-25"). true

> despachante:valid(cpf, "82520578025", true). true

> despachante:valid(cpf, "825.205.780-25", true). false
